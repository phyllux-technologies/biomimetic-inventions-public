# Contributing to David-Sproule-s-Inventions

This repo is primarily an archive. Contributions welcome for improvements (e.g., code optimizations, docs). Follow these steps:

1. Fork the repo.
2. Create a branch: `git checkout -b your-feature`.
3. Commit changes with clear messages.
4. Push and open a PR.
5. Respect the code of conduct: Be kind, no harassment.

For major changes, open an issue first.
