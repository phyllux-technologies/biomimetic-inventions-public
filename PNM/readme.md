# Phyllotactic Neural Meshing (PNM)

Bio-integrated neural interface using golden-angle (137.5°) electrode spacing. See root README for overview.

## Specs
- Array: 121 electrodes, r_i = R * √(i/120), θ_i = i * 137.5° mod 360°.
- Materials: Polyimide substrate, IrOx electrodes.
- Full details in /docs PDFs.
